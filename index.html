Como usar:

  • Toque em "Iniciar Câmera" e permita o acesso à câmera
  • Posicione o código dentro da área de digitalização
  • O sistema detectará automaticamente códigos de barras e QR codes
  • Use "Trocar Câmera" para alternar entre as câmeras
  • Toque em "Exportar para TXT" para salvar os dados

Problemas comuns:

  • Certifique-se de que está em HTTPS ou localhost
  • Verifique se concedeu permissão para usar a câmera
  • Atualize seu navegador para a versão mais recente
  • Reinicie o aplicativo se a câmera não responder
